# Kinirc modules
My kitnirc modules. May or may not work.


## op.py
Request OP from the bot, can also give OP to other users. Can set TOPIC as well.

Usage:
```
op [user(s)]
topic <topic string>
```

Logs chat activity to the log output from kitnirc as well as a csv-file.
